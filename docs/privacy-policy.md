# プライバシーポリシー

最終更新日: 2026年3月26日

合同会社ALBALIZE（以下「当社」）は、ZAIQUEST（以下「本アプリ」）におけるユーザーの個人情報の取り扱いについて、以下のとおりプライバシーポリシーを定めます。

## 1. 収集する情報

本アプリは以下の情報を収集します。

### アカウント情報
- メールアドレス（ログイン認証に使用）

### 利用データ
- お気に入り登録した商品情報
- 発注メモ（数量・メモ内容）

### 技術情報
- アプリのクラッシュレポート（エラー情報、デバイス情報、OS バージョン）

## 2. 情報の利用目的

収集した情報は以下の目的で利用します。

- ユーザー認証とアカウント管理
- お気に入り・発注メモ機能の提供
- アプリの不具合修正と品質改善
- サービスの安定運用

## 3. 第三者への提供

当社は以下のサービスを利用しており、これらのサービス提供者にデータが送信されます。

| サービス | 目的 | データ |
|---------|------|--------|
| Supabase | 認証・データベース | メールアドレス、利用データ |
| Sentry | クラッシュレポート | エラー情報、デバイス情報 |

上記以外の第三者に個人情報を提供することはありません。

## 4. データの保管

- ユーザーデータは Supabase のクラウドサーバーに保管されます
- ローカルデバイスにはキャッシュデータ（商品情報）が保存されます
- クラッシュレポートは Sentry のサーバーに保管されます

## 5. データの削除

ユーザーは以下の方法でデータを削除できます。

- **お気に入り・メモ**: アプリ内で個別に削除可能
- **アカウント全体**: 当社へメールでご連絡ください（info@albalize9.com）

アカウント削除時、関連する全てのデータ（お気に入り、メモ）も削除されます。

## 6. セキュリティ

- データベースへのアクセスは Row Level Security (RLS) で保護されています
- 通信は全て HTTPS で暗号化されています
- パスワードは Supabase の認証基盤でハッシュ化して保管されます
